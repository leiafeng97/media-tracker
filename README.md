# Personal Media Tracker

A simple static web app for tracking books, theatre, films, TV shows, and exhibitions.

## Use

Open `index.html` in a browser. Entries are saved to the browser's local storage, so they stay available on the same device and browser.

## Fields

- Title
- Format
- Date consumed
- Rating out of 5
- Vibe in 3 words
- What stayed with me
- Who I would recommend it to
- Similar works
- Content idea
- Whether I would revisit it
